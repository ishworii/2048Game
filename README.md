# 2048

Console version of the classic 2048 game.

## Description

This is a text-based implementation of 2048 that runs in the terminal.

## How to Run

Compile and run the Java source files from the src directory.

## How to Play

Use keyboard input to slide tiles in four directions. Tiles with the same number merge when they collide. The goal is to create a tile with the number 2048.
